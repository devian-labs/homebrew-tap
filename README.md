# homebrew-tap

Homebrew Tap for Devian Desktop.

## Installation

You can install Devian Desktop via Homebrew by running:

```bash
brew tap devian-labs/tap
brew install --cask --no-quarantine devian-desktop
```

Or installing directly via URL:
```bash
brew install --cask --no-quarantine devian-labs/tap/devian-desktop
```
